# Trading Mastery Pro

A free, self-paced technical analysis trading course covering Forex, Crypto, and Stocks.

## Courses Available
- **Technical Analysis Mastery** — 6 sections, 33 lessons, Beginner to Expert

## How to Run
Just open `index.html` in any browser. No server required.

## Structure
- `index.html` — Landing page
- `courses/technical-analysis/` — Full course content
- `courses/technical-analysis/section-1/` — Course Introduction (4 lessons)
- `courses/technical-analysis/section-2/` — Market Structure (6 lessons)
- `courses/technical-analysis/section-3/` — Supply & Demand / Order Blocks (6 lessons)
- `courses/technical-analysis/section-4/` — Mastering Liquidity (5 lessons)
- `courses/technical-analysis/section-5/` — Risk Management & Position Sizing (6 lessons)
- `courses/technical-analysis/section-6/` — Putting It All Together (6 lessons)
- `css/` — Shared styles (main.css, course.css)
- `js/` — Shared scripts (course.js)

## Topics Covered
Market Structure, Supply & Demand, Order Blocks, Liquidity, Risk Management, Trading Psychology, Position Sizing, Multi-Confluence Setups, BOS, CHoCH, Breaker Blocks, Stop Hunts, Institutional Trading

## Features
- Three difficulty levels per lesson (Beginner / Intermediate / Expert)
- Progress tracking saved to localStorage
- Sidebar search and lesson filtering
- Mark Complete with persistent progress bar
- Final quiz with auto-grading
- Confetti celebration on course completion
- Fully responsive (mobile-friendly)
